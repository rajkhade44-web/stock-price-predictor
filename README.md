# 📈 Stock Price Direction Predictor

## 🚀 Project Overview
This project predicts whether a stock price will go **UP or DOWN the next day** using Machine Learning.

---

## 🧠 Problem Statement
Predict stock movement direction with accuracy better than random (50%).

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- yFinance

---

## 📂 Project Structure
- data/raw → raw datasets
- data/processed → cleaned & feature data
- src/data → data processing
- src/features → feature engineering
- src/models → model training
- src/evaluation → evaluation

---

## ⚙️ Features Used
- Daily Return
- Moving Averages (5, 10)
- Volatility
- Volume Change
- Lag Features

---

## 🤖 Model Used
- Logistic Regression
- Random Forest Classifier

---

## 📊 Results
- Accuracy: ~0.46–0.52

⚠️ Note: Stock market prediction is inherently noisy and difficult.

---

## 🧠 Key Learnings
- Time-series data handling
- Feature engineering importance
- Model limitations in financial prediction

---

## 🚀 Future Improvements
- Use LSTM / Deep Learning
- Add technical indicators (RSI, MACD)
- Hyperparameter tuning
- Live prediction system
